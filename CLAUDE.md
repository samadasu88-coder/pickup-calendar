# お迎え予定表プロジェクト

このフォルダでは、毎月の送迎スケジュールを生成する。
ユーザーが勤務データを伝えたら、以下の手順で処理すること。

## 手順

1. ユーザーから月次データを受け取る
2. 判定ロジックに従って各日のスケジュールを決定
3. HTMLカレンダーを `YYYY-MM/お迎え表.html` に出力
4. Googleカレンダーの3つのカレンダーに予定を登録（前月分は削除してから）

## 名前の表記統一

担当者・子どもの名前は以下で統一する。揺れ（つー／通チャン／ユーナ／長女／次女 等）は使わない。

- **パパ** / **ママ** / **つーちゃん** / **じいじ**
- 子ども: **ゆうな**（小1・小学校）/ **ほのか**（年中・幼稚園）

## 月次データ（ユーザーから受け取る）

```
対象月: YYYY年M月
パパ: 0(休)/1(早１)/2(遅番) の数列（1日目から末日まで）
ママ: 0(休)/1(出勤) の数列（1日目から末日まで）
つーちゃん休み: 日付カンマ区切り（例: 4,9,13）
じいじ休み: 火水固定（変更があれば日付指定）
学校情報: 特殊日程があれば（園休み・午前授業・祝日など）
```

※パパの勤務表画像を渡された場合はOCRで読み取る。記号: 早１=1, ◯◎=2, /有=0, 下段「残」=2扱い。
※ママは出勤か休みかだけ判定。

## 家族と役割

| 名前 | 勤務 | 役割 | 制約 |
|------|------|------|------|
| パパ | シフト（早１/遅番/休み） | 全部できる | - |
| ママ | シフト（出勤/休み）。**土日祝は確定休み** | 全部できる | - |
| つーちゃん | 毎月指定 | 全部できる（朝もお迎えも） | - |
| じいじ | **火水固定休み** | 家でゆうなの帰りを待つだけ | **バス送迎不可（朝も帰りも）** |

### 優先順位
**パパ → ママ → つーちゃん → じいじ**

## 子ども

| | ゆうな（小1・小学校） | ほのか（年中・幼稚園） |
|--|---------------|---------------|
| 通学 | 徒歩 | バス（条件あり） |
| 預かり先 | キッズクラブ | どんぐり（延長保育） |

幼稚園バス: パパ/ママ/つーちゃんの誰かが休みのときだけ利用可。

## 判定ロジック

### パターン1: パパ/ママ/つーちゃんの誰かが休み
→ 休みの人を**全員**列挙する。複数人休みなら改行で並べる。
例: パパとママが両方休み → 「パパ休み」「ママ休み」を両方表示。詳細不要。

### パターン2: じいじだけ休み
- 朝: **朝どん**（ママがほのかを幼稚園に送る。「ママ」とは書かない）
- ゆうな（小学校）の帰り: 記載不要（じいじが家で待つ）
- ほのか（幼稚園）の帰り: パパがお迎え
  - パパ早１ → **どん17:30 パパ**
  - パパ遅番 → **どん18:30 パパ**
- ただし学校が休みの期間（春休み等）: じいじでは対応不可 → パターン3と同じ扱い

### パターン3: 全員仕事
- 朝: **朝どん**（通常時）/ **朝キッズ 朝どん**（学校が休みの期間のみ）
- 帰り:

| パパの勤務 | キッズクラブ | どんぐり |
|-----------|------------|---------|
| 早１ | キッズ17:00 パパ | どん17:30 パパ |
| 遅番 | キッズ18:00 ママ | どん18:30 パパ |

### パターン4: 土日祝
- **日曜**: 基本的に全員休みなので、パパ・ママ・つーちゃんの休み表示は不要（祝日名ラベルだけ）
- **土曜・平日の祝日**:
  - ママは確定休みなので表示しない
  - **パパが休み（0）** → 「パパ休み」を表示
  - **つーちゃんが休み** → 「つーちゃん休み」を表示
  - パパが仕事＆つーちゃん出勤なら空欄でOK
- 祝日名ラベル（憲法記念日 等）は曜日問わず表示

## 表記ルール

- パパ/ママ/つーちゃん休みの日 → 休みの人を全員「◯◯休み」で表示（複数人いれば複数行）
- 「歩いて帰宅」「◯◯が見る」等 → 書かない
- 施設お迎え → 「施設名+時間 担当者」の順（例: どん18:30 パパ）
- 朝の預け → 「朝キッズ」「朝どん」
- 朝どんを送るのはママ固定なので「ママ」とは書かない

## HTML出力仕様

- スマホで見やすいカレンダーグリッド（月〜日の7列）
- 各セルに日付 + 担当行（row-item）
- 土日祝は空セル（日付のみ）
- 下部に**施設利用サマリー**（キッズ/どんぐり別に朝・お迎えを日付でまとめた表）
- 下部に**家庭預かりサマリー**（通チャン預かり／ジージ預かりの該当日リスト。下記参照）
- 凡例
- CSSクラス: bg-papa, bg-mama, bg-tsuu, bg-jiiji, bg-off, bg-none
- 完成例: `2026-04/お迎え表.html`

## 家庭預かりサマリー（カレンダー下に表示）

カレンダー本体の予定としては書かず、下部サマリーに該当日リストを出す。

### つーちゃん預かり
- 条件: **パパ出勤 & ママ出勤 & つーちゃん休み**
- カレンダー本体は通常通り「つーちゃん休み」と表示。下のサマリーに該当日を一覧で出す。

### じいじ預かり（ゆうなのみ）
- 条件: **パパ出勤 & ママ出勤 & つーちゃん出勤 & 火曜or水曜**（じいじ火水休みのため在宅）
- カレンダー本体の予定は既存ロジックのまま（朝どん／どんお迎え等）。下のサマリーに該当日を一覧で出す。
- **ゆうな（小1・小学校）のみ対象**（小学校から徒歩で帰宅、じいじが家で待つ）。ほのか（年中）はどんぐり利用。

家庭預かりサマリーは **つーちゃん預かりとじいじ預かりを2カラム並び** で表示する。
HTMLの見出しは「つーちゃん預かり」「じいじ預かり」とだけ書く（条件説明・対象児童の補足は付けない）。

## Googleカレンダー

### カレンダーID
```
お迎え パパ: fa41ddc0faaf73e0a5eada89d706f64cb2472186477e337c668ff3849871523b@group.calendar.google.com
お迎え ママ: 0bc444caa792e29626e012dc6811e98555cef529fc6914f5662ac9500050fc6f@group.calendar.google.com
お迎え つー: 1e93543e644db22ac163df084fe4d00af489feebf01c69b8567e2d10dd48ec3a@group.calendar.google.com
```

### 登録ルール
- 終日予定で登録
- colorIdは不要（カレンダー自体の色で区別するため）
- 前月分がある場合は削除してから登録
- パパ担当の内容 → パパカレンダー、ママ担当 → ママカレンダー、つーちゃん → つーカレンダー
- gws CLIを使用: `gws calendar events insert --params '{"calendarId":"..."}' --json '{"summary":"...","start":{"date":"..."},"end":{"date":"..."}}'`
- 認証切れの場合: `gws auth login -s gmail,calendar` を案内

## 入力データの保存

HTML生成時に、受け取った勤務データを `YYYY-MM/input.md` に必ず保存する。
後から修正・再生成するときにこのファイルを参照する。

```markdown
# YYYY年M月 入力データ
対象月: YYYY年M月
パパ: 0/1/2 の数列（1日目から末日まで）
ママ: 0/1 の数列（1日目から末日まで）
つーちゃん休み: 日付カンマ区切り
じいじ休み: 火水固定（変更があれば日付指定）
学校開始: M/D
幼稚園開始: M/D
園休み: 日付カンマ区切り
祝日: 日付（名称）
備考: 特記事項
```
