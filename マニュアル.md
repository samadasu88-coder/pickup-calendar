# お迎え予定表マニュアル

## このツールでできること

毎月の勤務表をもとに、娘2人の送迎スケジュールを自動生成する。
キッズクラブ・どんぐり（延長保育）の申し込みに必要な情報がまとまる。

---

## 家族と役割

| 名前 | 勤務 | できること | できないこと |
|------|------|-----------|-------------|
| パパ | シフト制（早１/遅番/休み） | 全部 | - |
| ママ | シフト制（出勤/休み）。土日祝は確定休み | 全部 | - |
| つー | 休みの日を毎月指定 | 全部（朝もお迎えも） | - |
| じいじ | 火水固定休み | 家で小学生の帰りを待つ | バス送迎（朝・帰り両方） |

### 優先順位（誰が担当するか）

**パパ → ママ → つー → じいじ**

休みの人が複数いる場合、この順番で一番上の人が担当。

---

## 子どもの情報

| | 長女（小学校） | 次女（幼稚園） |
|--|---------------|---------------|
| 通学 | 徒歩 | バス（条件あり） |
| 預かり先 | キッズクラブ | どんぐり（延長保育） |

**幼稚園バスの条件**: パパ・ママ・つーの誰かが休みのときだけ利用可。じいじだけ休みの日はバス不可。

---

## 判定ルール

### パターン1: パパ/ママ/つーの誰かが休み

→ **「◯◯休み」とだけ書く。** 朝も帰りもその人が対応。

### パターン2: じいじだけ休み

| | 小学校 | 幼稚園 |
|--|--------|--------|
| 朝 | 歩いて登校（記載不要） | ママが送り → **朝どん** |
| 帰り | 歩いて帰宅、じいじが家で待つ（記載不要） | パパがお迎え |

幼稚園のお迎え時間はパパの勤務で決まる:
- パパ早１ → **どん17:30**
- パパ遅番 → **どん18:30**

### パターン3: 全員仕事

| | 小学校 | 幼稚園 |
|--|--------|--------|
| 朝 | 歩いて登校（通常時）/ **朝キッズ**（春休み等） | **朝どん** |
| 帰り | パパの勤務で変わる（下表） | パパの勤務で変わる（下表） |

| パパの勤務 | キッズクラブ | どんぐり |
|-----------|------------|---------|
| 早１ | キッズ17:00 パパ | どん17:30 パパ |
| 遅番 | キッズ18:00 ママ | どん18:30 パパ |

### パターン4: 土日祝

何も書かない。ママが確定で休み（パパも日曜は基本休み）。

## パパの勤務記号（勤務表から読み取る）

| 記号 | 意味 | コード |
|------|------|--------|
| 早１ | 早番 | 1 |
| ◯・◎ | 遅番 | 2 |
| /・有 | 休み | 0 |

※勤務表は2行構成。下段に**「残」**がある日は遅番（2）扱い。

## 毎月の作業手順

### 1. データを準備する

| 項目 | 入力形式 | 例 |
|------|---------|-----|
| パパ勤務 | 1日目から順に 0/1/2 の数列 | `210121020120...` |
| ママ勤務 | 1日目から順に 0/1 の数列 | `101001001000...` |
| つー休み | 日付のカンマ区切り | `4,9,13,14,30` |
| じいじ休み | 基本は火水固定。変更あれば指定 | 火水固定 |
| 学校カレンダー | 開始日・園休み・祝日 | 小:7日〜 / 幼:8日〜 / 園休み:9 / 祝日:29 |

### 2. Claude Codeに渡す

「お迎え表を作って」と伝え、上記データを入力する。

### 3. 出力されるもの

1. **HTMLカレンダー** — スマホで確認・スクショ用
2. **施設利用サマリー** — キッズ・どんぐりの申し込みに直接使える
3. **Googleカレンダー** — 3つのカレンダーに自動登録、家族と共有

---

## 出力の見方

### カレンダーの表記

| 表記 | 意味 |
|------|------|
| パパ休み / ママ休み / つー休み | その人が全部対応。詳細不要 |
| 朝どん | 朝、幼稚園をどんぐりに預ける |
| 朝キッズ | 朝、小学校をキッズクラブに預ける（春休み等のみ） |
| キッズ17:00 パパ | キッズクラブにパパが17:00にお迎え |
| どん18:30 パパ | どんぐりにパパが18:30にお迎え |

### 色分け

| 色 | 担当 |
|----|------|
| 青 | パパ |
| ピンク | ママ |
| 緑 | つー |
| オレンジ | じいじ |

---

## Googleカレンダーの共有

### カレンダー構成（3つ）

| カレンダー名 | 色の設定 | 内容 |
|-------------|---------|------|
| お迎え パパ | 青 | パパ担当の予定（お迎え時間、パパ休み） |
| お迎え ママ | ピンク | ママ担当の予定（朝どん、ママ休み、キッズお迎え） |
| お迎え つー | 緑 | つー担当の予定（つー休み） |

### 共有手順

1. Googleカレンダーの設定 → 各カレンダーの「特定のユーザーと共有」
2. 相手のGmailアドレスを追加
3. 共有された側で各カレンダーの色を設定してもらう（青・ピンク・緑）
